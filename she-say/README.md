# 她说 (she-say)

一个基于 **cangjie-skill** 框架开发的性别议题分析 Skill，帮助你理性拆解性别热点、情感争议和人际观念冲突。

## ✨ 功能特点

- **双视角平衡解读**：兼顾男性、女性视角，避免极端立场
- **理论按需辅助**：融合《厌女》《第二性》等女性主义理论，但不强行套书
- **落地实用**：提供非暴力沟通话术和应对建议
- **温和理性**：拒绝标签化和人身攻击，以理解替代对立

## 🚀 安装使用

### 方法一：下载使用

1. 点击 GitHub 页面右上角的 **绿色按钮 "Code"**
2. 选择 "Download ZIP" 下载整个仓库
3. 解压后，将 `she-say` 文件夹复制到你的 Agent 的 `skills/` 目录下
4. 重启 Agent
5. 在对话中输入：`she say 使用技能：你的问题`

### 方法二：Git 克隆（推荐）

```bash
git clone https://github.com/YoJeari/她说.git
```

然后将 `she-say` 文件夹复制到你的 Agent 的 `skills/` 目录。

## 📖 使用示例

**输入：**
```
she say 使用技能：社会对男生晚婚很包容，为什么对大龄女生很苛刻？
```

**输出：**
会从客观事实、多元视角、理论理解、结论和沟通建议等方面进行分析。

## 🙋‍♀️ 常见问题

**Q：这个 Skill 会有偏见吗？**

**A：** 我们努力保持中立客观，兼顾不同视角，避免极端立场。所有分析基于事实和理论，不进行道德审判。

**Q：如何更新到最新版本？**

**A：** 如果你用 Git 克隆，只需在 `she-say` 文件夹中运行：
```bash
git pull
```

**Q：可以在哪些 Agent 中使用？**

**A：** 支持任何兼容 cangjie-skill 框架的 Agent。

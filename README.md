## 为什么做这件事

当代性别议题的讨论，常常陷入两个极端：要么是情绪化的对立骂战，要么是学术化的理论独白。

一边是社交媒体上无休止的男女互撕——每个人都在说话，但没有人真正在听；标签满天飞，但问题从未被拆解。另一边是书架上的女性主义经典——被读过、被划线、被收藏，但在你最需要回应"为什么女生素颜就被指指点点"的时候，那些理论一句都想不起来。

这中间缺了一步：**把书里真正有用的视角，变成你在现实场景中能调用的分析工具。**

she-say不是教你怎么吵架的，也不是教你怎么站队的。它提供的是一种看问题的方式——让你看到那些"理所当然"背后的结构，看到“大家都这么觉得”背后的盲区。但如果你只能用“打拳”或"直男癌"来表达，那理论就白读了。

我们希望她说能做到的是：**不管你是女性还是男性，面对性别争议时，能多一个思考的角度，而不是多一个伤害他人的武器。**

## 它解决了什么问题

**看了很多女性主义的内容，但遇到现实争论还是只会生气**

刷到性别争议帖，你知道哪里不对，但说不清楚到底哪里不对。理论在脑子里是散的，没法在关键时刻组织成有力的分析。

→ she-say 借助 cangjie-skill 把 经典著作的方法论蒸馏成结构化分析器，你提问，它按"事实→视角→理论→结论→话术"的路径帮你拆解，不是给你一个结论，而是让你看到全貌。

**讨论性别问题容易变成情绪对立**

一开口就被贴标签——"你打拳""你厌女"。双方都在自说自话，没有人真正在拆问题。

→ she-say 坚持双视角输出，同时呈现女性和男性的认知逻辑，并点破双方的隐藏前提，让你看到对方为什么那么想。

**理论太学术，和现实接不上**

《第二性》讲"他者化"，《厌女》讲"双重标准"——概念都懂，但怎么对应到"为什么职场对女性穿着要求更严"这种具体问题上？

→ she-say 只在真正相关时才调用理论，用大白话解释，不搞学术黑话。理论是工具，不是主角。

**知道问题在哪，但不知道怎么回应**

被亲戚催婚、被同事说"女生不用那么拼"、被伴侣说"你想太多了"——心里不舒服，但不知道怎么体面地回应。

→ she-say 提供三个层级的沟通话术：温和沟通句（化解对立）、得体回击句（体面不引战）、落地行动（具体可执行的建议）。

## 它是怎么工作的

she-say 使用 **"现实优先、理论辅助"** 的分析流水线，把你的问题从混沌变成清晰：

1. **厘清事实** — 先用常识、法律、社会规则，拆透这件事的本质是什么
2. **多元视角** — 呈现不同性别的真实认知逻辑，点破隐藏前提
3. **论点校验** — 检查每个视角下的论点是否真的支持该视角，避免逻辑错位
4. **理论辅助（按需）** — 只有当理论能让你看得更深时才调用，否则不硬套
5. **多维度补充** — 补充阶层、城乡、代际等视角，打破二元局限
6. **明确结论 + 沟通话术** — 给出有判断的结论，同时提供可使用的回应方式

整个过程中，女性主义经典著作作为知识库在后台待命——你不需要知道该用哪本书，她说是按需调用的。

## 扩展知识库

你可以往 `books/` 文件夹里添加更多书籍，she-say 会自动识别并按需调用。

每本书需要按以下结构放入：

```
books/<你的书籍ID>/
├── BOOK_OVERVIEW.md      # 整书概览
├── INDEX.md              # 全文索引 + skill关系图
├── counter-examples.md   # 反例库
└── analyzers/            # 分析器文件夹
    └── [<章节>_<名称>.md]
```

添加完成后，在 `SKILL.md` 的知识库表格中追加一行即可。

**怎么蒸馏一本书？** 推荐使用 [cangjie-skill](https://github.com/kangarooking/cangjie-skill) 框架——它可以把任何一本书的方法论蒸馏成结构化的分析器，直接放入 she-say 的 `books/` 文件夹就能用。

## 常见问题

**她的立场是什么？是女权吗？**

she-say 提供的是分析，不是立场。它做的事情是把一个问题从多个角度拆开，让你看到全貌。女性主义理论是它的工具之一，不是它的结论。如果你读完分析后觉得"原来对方是这么想的"，那它的目的就达到了。

**输出会不会偏向女性？**

she-say 坚持双视角输出——男性和女性的认知逻辑都会被呈现。它不会替你判断谁对谁错，但会指出双方各自看不到的盲区。偏向任何一方都不是目的，让你看到更完整的图景才是。

**理论会不会很难懂？**

不会。理论只在真正相关时才出现，而且一定用大白话解释。你不需要读过《第二性》也能理解"他者化"在说什么。如果理论不能让你看得更清楚，它就不会出现。

**遇到极端言论怎么办？**

she-say 不参与骂战。对于极端引战、性别辱骂、人身攻击的输入，它会温和婉拒。它的价值在于拆解问题，不是升级冲突。

**男生也可以用吗？**

当然。性别议题不是某一性别的专属话题。如果你是一个想理解对方视角的男生，或者你在某个争论中感到困惑she-say 同样能帮到你。理解不等于认同，但理解是对话的前提。

**和直接问 AI 有什么区别？**

直接问 AI，你得到的是一个回答。问 she-say ，你得到的是一套分析——事实是什么、各方怎么想、隐藏前提是什么、理论怎么看、局限在哪里、怎么回应。它不急着给你结论，而是先让你看清问题。
